# qqmail-client
qq企业邮箱的桌面版客户端
***
## 功能
![](image/screenshot.png "截图")
> + 新邮件推送提醒
> + 未读邮件标记

## 安装
```Bash
    # Clone this repository
    git clone https://github.com/qq181958825/electronic-qqmail
    # Go into the repository
    cd electronic-qqmail
    # Install dependencies
    npm install
    # Run the app
    npm start
```
